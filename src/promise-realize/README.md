## 说明
将来可能成为主流的async-await，以及曾经火过一把的generator + co，这些都是需要编译器或者解释器级别的支持才能使用。


而Promise，是完全可以利用语言已有特性，作为一个库来实现！即使在非常原始的JS运行环境，你也可以自己实现一个Promise，而不需要等待其他人的帮助。

Promise还是独立于语言的，如果你要给另外一种编程语言实现Promise，只要照葫芦画瓢就行了。
